# ダッシュボード

> **ひとことで：** 検出された脅威の全体像を一目で把握し、対応が必要なものから順番に確認できる画面です。

## 画面構成

### サマリーカード

画面上部に3枚のカードで現在の状況を表示します。

| カード | 色 | 表示内容 |
|--------|-----|---------|
| 🔴 **対応が必要** | 赤 | 脅威が確認され、削除申請などのアクションが必要な件数 |
| 🟡 **確認待ち** | 黄 | 新規検出されたが、まだ調査・判定が完了していない件数 |
| 🟢 **対応済み** | 緑 | 削除申請が完了した、または安全と判定された件数 |

各カードをクリックすると、該当ステータスで脅威一覧がフィルタリングされます。

### 脅威一覧テーブル

検出された全ドメインを一覧表示します。

| 列 | 内容 |
|----|------|
| ドメイン名 | 検出されたドメイン |
| ブランド | 対象ブランド名（複数ブランド管理時） |
| リスクスコア | 0〜100のスコア（色分け表示） |
| ステータス | new_domain / confirmed_threat / takedown_sent / resolved 等 |
| 検出日 | 最初に検出された日時 |

- **デフォルトソート**: リスクスコア降順（高リスクが上）
- **ブランドフィルター**: ドロップダウンで特定ブランドに絞り込み

### レイヤー2展開パネル

テーブルの各行を展開すると、詳細情報を確認できます：
- AI分析の判定理由
- 脅威カテゴリ
- 最終調査日時

::: tip 操作のコツ
まず 🔴「対応が必要」をクリックして高リスクの脅威に絞り込み、上から順に詳細を確認→削除申請に進むのが効率的です。
:::
