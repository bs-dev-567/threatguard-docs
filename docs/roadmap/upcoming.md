# 今後の開発予定

> 以下は今後のリリースで追加予定の機能です。優先度や時期は変更される場合があります。

## 近日追加予定

### 📧 削除申請の強化

| 機能 | 説明 |
|------|------|
| **リマインド自動送信** | 一定期間回答がないレジストラへの自動リマインド |
| **送信先の変更** | 却下時に別のabuse窓口（ホスティング事業者等）へ再申請 |
| **abuse連絡先判定の精緻化** | WHOIS/RDAP以外のソース（各レジストラのWebページ等）からの連絡先取得 |
| **PDFエクスポート** | 削除申請の内容をPDF形式でダウンロード |

### 📊 レポート・エクスポート

| 機能 | 説明 |
|------|------|
| **定期レポート** | 週次・月次の脅威検出・対応状況レポートの自動生成 |
| **CSV/PDFエクスポート** | 脅威一覧・削除申請履歴のエクスポート |
| **金融庁向けレポート** | セキュリティインシデント報告に準拠したフォーマット |

### 🔍 検知の拡張

| 機能 | 説明 |
|------|------|
| **SNS監視** | X（Twitter）、Telegram、Discord上の偽公式アカウント検知 |
| **偽エアドロップ・偽IDOサイト検知** | 暗号資産特有の詐欺パターンへの対応 |
| **ウォレット接続フィッシング検知** | MetaMask等を模倣したサイトの特定 |

## 中長期の計画

### 🔗 外部連携

| 機能 | 説明 |
|------|------|
| **JPCERT/CC連携** | フィッシング情報の共有とテイクダウン連携 |
| **Google Safe Browsing報告** | 検出した脅威をGoogleの安全ブラウジングリストに報告 |
| **API公開** | 外部システムとの連携用REST API |

### 🛡️ 高度な分析

| 機能 | 説明 |
|------|------|
| **リアルタイムCT監視** | CT Logのストリーミング監視（WebSocket） |
| **ダークウェブ監視** | Torネットワーク上のブランド言及を検出 |
| **マーケットプレイス監視** | ECサイト上の偽造品・ブランド侵害の検知 |

### 📱 利便性向上

| 機能 | 説明 |
|------|------|
| **モバイル対応UI** | スマートフォンからの確認・承認操作 |
| **Teams連携** | Microsoft Teams へのアラート通知 |
| **多言語対応** | 英語UIの提供 |
